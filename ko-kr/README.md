# hooh's Modding Tool

## Welcome

hooh의 모딩 툴에 오신 것을 환영합니다.

이곳에서 허니셀렉트 2 또는 AI 소녀의 모딩 방법을 배우실 수 있습니다.

## Before doing something

이 페이지는 블렌더 기초 같은 모든 기본적인 기술을 포함하고 있지 않습니다.

모딩을 위해서는 기본적은 3D 그래픽에 대한 지식과 유니티 엔진이 어떻게 작동하는가에 대한 방법을 알아야 시작할 수 있습니다.
